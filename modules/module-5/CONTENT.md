## Part 3: Mini-Project

### Choose One Project

---

### Project A: Design Your Personal AI Agent

**Task:** Design an AI agent that would help you in your current role or life.

**Deliverables:**
1. **Agent SOUL.md** — Who is this agent? What are its values?
2. **Agent AGENTS.md** — What does it do? Walk through one specific workflow
3. **Tools.md** — What tools does it need?
4. **Use Case Description** — Write 2 concrete examples of how it would help you

---

### Project B: AI Strategy Brief

**Task:** Write a 1-page AI strategy brief for your organisation or a fictional company.

**Deliverables:**
1. Current state assessment (maturity level + existing tools)
2. Top 3 use cases (with impact/effort rating)
3. 90-day roadmap
4. Ethical considerations checklist
5. One quick win you could implement this week

---

### Project C: The Agentic Workflow

**Task:** Design a multi-step workflow automated by AI agents.

**Scenario Options:**
- A travel planning agent that books complete trips
- A research agent that produces a weekly industry report
- A sales agent that qualifies leads and books meetings
- Your own idea (approved by coach)

**Deliverables:**
1. ASCII diagram of the workflow
2. List of agents involved (and their specialties)
3. Tools each agent uses
4. How memory is shared between agents
5. What happens when something goes wrong (error handling)
